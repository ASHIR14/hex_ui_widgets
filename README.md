## Features

- Customizable buttons of multiple types and size.
- Swipe button
- Custom TextFormField
- Custom Toast Widget
- Bottom Navigation Bar

## Getting started

In your flutter project add the dependency:

```yaml
dependencies:
  hex_ui_widgets: any
```

Import the package:

```dart
import 'package:hex_ui_widgets/hex_ui_widgets.dart';
```
